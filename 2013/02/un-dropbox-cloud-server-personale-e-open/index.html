Un “dropbox” (cloud server) personale e open

Oggi il lavoro al computer non è più limitato ad una singola macchina. Le persone hanno la necessità di condividere i loro dati con diversi dispositivi, come computer fissi, portatili, tablet, telefonini. Per questo una cosiddetta soluzione di tipo cloud è assai utile.

Molti si servono di strumenti o siti come Dropbox, Ubuntu One, Google Drive, box.net, eccetera. Queste soluzioni, però, non sono adeguate per la tutela della tua privacy e fanno perdere il controllo sui dati condivisi con più macchine.

Spesso molti clienti vorrebbero avere i dati sempre disponibili nei loro dispositivi (il cloud, la nuvola) ma al contempo non lasciarli in server “di altri” (dropbox, box.net, google drive,…).

Vorrebbero una sorta di dropbox personale, sul proprio server.

In questo post vi segnalo una interessante soluzione open source, scritta in php, con client per Windows, Mac, Linux, Android, iOS,… integrabile, scalabile.

Il progetto si chiama OwnCloud e trovate tutte le informazioni al link http://owncloud.org, è stato avviato nel gennaio 2010 da Frank Karlitschek, del gruppo di sviluppo di KDE per creare una alternativa libera ai cloud provider commerciali.

Il progetto è basato su PHP e un DB tra SQLite, MySQL o PostgreSQL, quindi ownCloud può essere eseguito su tutte le piattaforme che soddisfano questi requisiti.

Il software è rilasciato con licenza AGPLv3.

Le caratteristiche dettagliate le trovate al link http://owncloud.org/features/. In sintesi:

  • Client multipiattaforma
  • Server multipiattaforma
  • Interfaccia web
  • Drag e drop per upload e operazioni comuni
  • Gestione delle diverse versioni dei file
  • Visualizzatore online per documenti office-like
  • Condivisione di singoli file
  • Notifiche su file condivisi, modificati,…
  • Autenticazione integrabile con LDAP, Active Directory,…
  • Backup e task
  • Crittografia dei file
  • Log

Le istruzioni per l’installazione le trovate al link

http://doc.owncloud.org/server/4.5/admin_manual/installation.html.

A mio avviso un progetto molto interessante da tenere d’occhio.
