Monitor informativi

Alcuni clienti mi hanno chiesto la realizzazione di monitor informativi. Per intenderci far vedere immagini, pagine web o altro in alcuni monitor sparsi per la sede.

Finora avevo sempre utilizzato soluzioni self-made ma da un po’ di tempo queste non incontrano più le mie necessità. Per questo ho deciso di cercare nel web e fare alcune prove. Alla fine ho trovato quello che cercavo: un sistema semplice, flessibile e open source che mi permette di erogare contenuti testuali, filmati, immagini, pagine web, video, filmati flash, ticket rss,…

L’idea è semplice: un server web con php e mysql dove depositare i contenuti e X client (i meno esigenti possibili in termini hardware e software) collegati a dei monitor TV standard. Dall’interfaccia web comandare la visualizzazione su tutti i monitor senza toccare nulla.

Prima di tutto un termine cruciale: “digital signage”. Se dovete fare ricerche su questo tema usate queste keywords su google.

Passiamo al concreto.

Xibo (pronunciato eX-E-bO) è una soluzione digital signage open source, multi-display, multi-zona, pienamente schedulabile controllabile da un’interfaccia web a gestione centralizzata.

Xibo è rilasciato sotto licenza Affero GNU Public License v3 o successive.

Si appoggia a php, mysql e sono disponibili client di visualizzazione per windows e Linux.

La forza del sistema sta nella gestione di diversi viewer che possono essere anche a notevole distanza tra loro, vi è la possibilità di avere diversi formati ed anche di pianificare le visualizzazioni a piacimento durante diversi periodi della giornata.

Il client esiste per Windows XP e per Ubuntu.
