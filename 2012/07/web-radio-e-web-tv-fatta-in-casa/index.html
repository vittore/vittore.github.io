Web radio e web tv fatta in casa

Una web radio (o web tv) è un server web che dispensa agli utenti collegati (via internet)

un flusso audio (o video) che si ascolta mentre si è collegati ad

internet, questo tipo di flusso prende il nome di streaming. Per funzionare, la

web radio, ci deve essere qualcuno che trasmette questo flusso da un

computer.

Volete crearvi una web radio o una web tv per trasmettere i vostri contenuti? Se siete insegnanti potete trasmettere le vostre lezioni, in diretta oppure in differita.

Eccovi due guide interessanti:

  • Web Radio e PodCast ver. GNU/Linux (ODT)  (PDF)
  • Web Radio e PodCast ver. Windows (ODT)  (PDF)
