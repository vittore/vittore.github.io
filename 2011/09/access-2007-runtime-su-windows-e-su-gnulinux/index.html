Access 2007 runtime su Windows e su GNU/Linux

Oltre alla schiavitù mentale, (per dirla alla Bob Marley in Redemption song) le obiezioni che i detrattori dei sistemi GNU/Linux pongono al passaggio dal software proprietario (Windows) a quello libero sono sostanzialmente 2: non gira Autocad e non gira Access!

Come mostrato in

http://www.comunecampagnano.it/gnu/mini-howto/Chi_ha_detto_che_Linux_non_ha_un_CAD/Chi_ha_detto_che_Linux_non_ha_un_CAD.pdf

il problema del CAD è una bufala.

Rimane il problema di Access inteso come applicazione finita con menu, maschere, macro, report eccetera perché se l’interesse è il “dato” questo (le tabelle) può benissimo essere importato dentro Base di OpenOffice.org o meglio ancora in LibreOffice.

Va detto, per inciso,  che tramite wine si può installare benissimo MS Access su GNU/Linux, ma questo non risolve il problema del costo della licenza.

Le buone notizie sono tre:

  1. Microsoft permette ora di scaricare il runtime di Access 2007 e 2010 gratuitamente
  2. Con il runtime di access 2007 si possono aprire e utilizzare MDB fatti con Access 2003
  3. Tramite wine si può installare Access 2007 runtime anche su GNU/Linux

Il runtime di Access permette di utilizzare il database come utente finale e non come programmatore, ma se qualche genietto, in una azienda o ufficio pubblico, ha creato un MDB fondamentale che deve essere utilizzato da decine e/o centinaia di utilizzatori si capisce facilmente il risparmio dovuto al non acquisto delle licenze d’uso di MS Access.

WINDOWS

Una volta scaricato il file AccessRuntime.exe di circa 50 MB

e installato troveremo il runtime dentro C:\Programmi\Microsoft Office\Office12\ come mostrato nella figura sottostante

Quando tenteremo di aprire il file clienti.mdb (creato con Acess 2003) con il tasto destro e scegliendo apri con access runtime apparirà un avviso di protezione come mostrato in figura

e dopo aver cliccato su apri, si aprirà l’applicazione che tramite la macro Autoexec mostrerà una maschera basata su una Query

GNU/Linux

Una volta scaricato il file AccessRuntime.exe di circa 50 MB, reso eseguibile modificandone le proprietà, e installato troveremo il runtime dentro //home/nomeutente/.wine/drive_C/Programmi/Microsoft Office/Office12\

Quando tenteremo di aprire il file clienti.mdb (creato con Acess 2003) con il tasto destro e scegliendo apri con access runtime apparirà un avviso di protezione. e dopo aver cliccato su apri, si aprirà l’applicazione che tramite la macro Autoexec mostrerà una maschera basata su una Query
